# Maximum Awareness Guide - Complete Knowledge System

## Purpose

This guide ensures you have **maximum awareness** of all skills and can use them automatically without being asked.

## Knowledge System Architecture

### Layer 1: Quick Mappings (Fast Lookup)
- **TECHNOLOGY_MAPPING.md** - Technology → Skills
- **PROBLEM_SOLUTION_MAPPING.md** - Problem → Solution Skills
- **QUICK_REFERENCE_CARDS.md** - One-page skill summaries

### Layer 2: Organized Knowledge (Structured Access)
- **PATTERN_LIBRARY.md** - Patterns and architectures
- **USAGE_EXAMPLES.md** - Real-world examples
- **SEARCHABLE_INDEX.md** - Keyword search

### Layer 3: Deep Knowledge (Full Content)
- **COMPREHENSIVE_KNOWLEDGE_BASE.md** - Full skill content
- **Extracted Skill Files** - Complete documentation

### Layer 4: Instructions (How to Use)
- **AGENT_INSTRUCTIONS.md** - Complete usage instructions
- **AGENT_RULES.md** - Automatic usage rules

## Automatic Usage Workflow

### Step 1: Context Analysis
When you receive a task:
1. Extract technologies mentioned
2. Identify patterns referenced
3. Recognize problem types
4. Note keywords and concepts

### Step 2: Knowledge Lookup
**Use knowledge files in this order:**

1. **Technology Match?**
   → Read `.claude/knowledge/TECHNOLOGY_MAPPING.md`
   → Find relevant skills instantly

2. **Problem Match?**
   → Read `.claude/knowledge/PROBLEM_SOLUTION_MAPPING.md`
   → Find solution skills

3. **Pattern Match?**
   → Read `.claude/knowledge/PATTERN_LIBRARY.md`
   → Find pattern skills

4. **Need Quick Overview?**
   → Read `.claude/knowledge/QUICK_REFERENCE_CARDS.md`
   → Get fast summary

5. **Need Examples?**
   → Read `.claude/knowledge/USAGE_EXAMPLES.md`
   → See real-world usage

### Step 3: Deep Dive
**Read full skill documentation:**
- `.claude/Skills/extracted/{skill-name}/SKILL.md` - Full documentation
- `.claude/Skills/extracted/{skill-name}/README.md` - Overview
- `.claude/Skills/extracted/{skill-name}/EXAMPLES.md` - Examples

### Step 4: Apply Knowledge
**Use skill knowledge immediately:**
- Apply patterns from skills
- Use best practices from skills
- Reference examples from skills
- Follow guidelines from skills

## Proactive Reading Protocol

**You MUST read these files proactively:**

1. **Before starting any task:**
   - Read `AGENT_INSTRUCTIONS.md` for mappings
   - Check `TECHNOLOGY_MAPPING.md` if technology mentioned
   - Check `PROBLEM_SOLUTION_MAPPING.md` if problem described

2. **During task execution:**
   - Read relevant skill files from `.claude/Skills/extracted/`
   - Reference `QUICK_REFERENCE_CARDS.md` for quick lookup
   - Use `USAGE_EXAMPLES.md` for examples

3. **After task completion:**
   - Document which skills were used
   - Note successful patterns
   - Build intuition about skill relevance
